# xortools
This is a colllection of C++ and Python tools used to carry out the research reported in

Optimization by Decoded Quantum Interferometry
Stephen P. Jordan, Noah Shutty, Mary Wootters, Adam Zalcman, Alexander Schmidhuber, Robbie King, Sergei V. Isakov, Tanuj Khattar, and Ryan Babbush
https://arxiv.org/abs/2408.08292

A detailed catalog of the programs contained in this collection and instructions for how to compile and use them is given in guide_to_xortools.pdf.
